# Chess Resources

## Knowledge

- [Chess.com Lessons Guide](https://www.chess.com/lessons)
  Premium learning paths. Use for: Structured study courses on openings, middlegames, and endgames.
- [Chess.com Puzzles](https://www.chess.com/puzzles)
  Unlimited daily puzzle sets. Use for: Building board vision and tactical pattern recognition.
- [ChessKid Youtube Channel](https://www.youtube.com/@ChessKidOfficial)
  Fun, engaging, and kid-friendly chess instruction. Use for: Basic concepts explained with storytelling.
- [GothamChess Youtube Channel](https://www.youtube.com/@GothamChess)
  High-energy and entertaining chess analysis and beginner guides. Use for: Visualizing openings (e.g., "10-Minute Openings") and learning from mistake analysis.

## Wisdom (Communities)

- [Chess.com Forums](https://www.chess.com/forum)
  Community forums for game feedback and advice. Use for: Asking general questions and sharing games.
- [r/chessbeginners](https://reddit.com/r/chessbeginners)
  Friendly, beginner-focused chess discussion on Reddit. Use for: Finding advice on breaking rating plateaus.
